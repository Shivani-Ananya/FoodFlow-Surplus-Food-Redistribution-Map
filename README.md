# FoodFlow-Surplus-Food-Redistribution-Map
FoodFlow is a frontend concept platform that visualizes surplus food locations and hunger zones in real time using an interactive map. The project addresses the growing issue of food waste from restaurants, hostels, and college canteens while many communities nearby struggle to access meals

By mapping both food providers and demand locations, the system helps volunteers and organizations quickly identify where food can be redistributed.

This project currently serves as a frontend demonstration prototype that showcases how a data-driven map interface could support smarter food redistribution in urban areas.
